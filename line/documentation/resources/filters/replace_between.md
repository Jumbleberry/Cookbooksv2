# Examples for the replace_between filter

## Example 1: Original file

```text
line1
line2
line3
```

## Example 1: Output file

```textruby
line1
rep1
rep2
line3
```

## Example 1: Filter

```ruby
replines = "rep1\nrep2\n"
filter_lines '/example/replace_between' do
 filters(replace_between: [/^line1$/, /^line2$/, replines])
end
```
